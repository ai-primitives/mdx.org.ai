# mdx.org.ai

[mdx.org.ai](https://mdx.org.ai) provides @context for MDX files to have a specific @type that represents what it is. 

For example, a @type could be:

 - Page
 - Component
 - BlogPost
 - Documentation
 - LandingPage
 - Website
 - API
 - App
 - Function
 - Workflow
 - Agent
 - Tool
